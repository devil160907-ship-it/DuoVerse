# DuoVerse - Virtual Meeting Space for Couples

![DuoVerse](static/icons/icon-512.png)

DuoVerse is a private, secure virtual meeting platform designed specifically for couples. Create private rooms with scheduled meeting times, share messages, photos, and watch YouTube videos together in real-time.

### Core Features
- **Private Meeting Rooms**: Create unique rooms with scheduled dates and times
- **Join Request System**: Partners must request and be approved to join
- **Real-time Chat**: Instant messaging with image sharing
- **Photo Gallery**: Capture and share moments with password-protected gallery
- **YouTube Watch Together**: Synchronized video watching with playlist support
- **QR Code Sharing**: Easy room sharing via styled QR codes
- **Join Request Management**: Accept/reject participants with full history tracking

### Technical Features
- **Progressive Web App (PWA)**: Installable on mobile devices
- **Offline Support**: Basic offline functionality
- **Responsive Design**: Works on all devices
- **Real-time Updates**: Live chat and video synchronization
- **Environment Auto-detection**: Development vs Production
- **SQLite/MySQL Support**: Adapts to environment
